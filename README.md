# SamsTrack---Attendance-Management-System
SamsTrack is a lightweight and efficient Attendance Management System built using React.js for the frontend and Spring Boot for the backend.
It simplifies attendance tracking for students/employees with an easy UI, secure APIs, and smooth user experience.

Features

✔️ User Authentication (Login/Register)

✔️ Mark Attendance (Daily/Session-wise)

✔️ View Attendance Reports

✔️ Admin Panel for Managing Users

✔️ REST API Integration using Spring Boot

✔️ Responsive UI with React.js

✔️ CRUD Operations for Users & Attendance Records

Tech Stack
Frontend
React.js
Axios
HTML5 + CSS3
Bootstrap / Material UI (optional)

Backend
Spring Boot
Spring Web
Spring Data JPA
MySQL Database
